# Collaborator Documentation

This folder contains safe collaboration notes for the SiteSpy Expo React Native Firebase project.

- `collaborators-sanitized.md`: safe collaborator contact table.
- `task-allocation.md`: mobile and Firebase task ownership.
- `git-workflow.md`: branch and pull request rules.
- `frontend-team.md`: mobile UI responsibilities.
- `backend-team.md`: Firebase, data, quality, and deployment responsibilities.

Raw collaborator files with passwords or private login values must remain ignored.
