# Mobile UI Track

The mobile UI track owns the React Native user experience for Android portrait use.

Responsibilities:
- Auth screens and validation states.
- Navigation shell and bottom tabs.
- Dashboard and project history screens.
- Manual estimation forms and BOQ summary presentation.
- Image-assisted record screen layout.
- Profile, settings, spacing, touch targets, and Expo Go screenshots.
